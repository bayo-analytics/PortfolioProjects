#COVID-19 Data Analysis (SQL & Tableau)

##Project Overview

This project analyzes global COVID-19 data to uncover trends in confirmed cases, deaths, and vaccinations across contries and over time. SQL was used for data cleaning, transformation, and analysis, while Tableau was used to build an interactive dashboard for visualization.

##Tools & Technologies

- SQL
- Tableau
- Excel/CSV

##Dataset
The dataset in use contains detailed information about COVID-19, sourced from "Our World in Data". It encompasses a diverse range of data points related to the cases, deaths, vaccination data, countries, and date information of COVID-19. The dataset covers the total cases, total deaths, death percentage, and percentage population infected across all countries. This dataset serves as an invaluable resource for individuals seeking to practice data analysis techniques.

##SQL Analysis

SQL was used to:
- Clean and prepare the dataset for analysis
- Calculate total cases, total deaths, death percentages, and percentage population infected
- Analyze infection rates by country relative to population
- Track vaccination progress
- Create views to support Tableau visualization
